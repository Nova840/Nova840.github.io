Margin, Border, and Padding in CSS.

2/28/16

Margin, border, and padding are important to understand if you're using CSS. In this blog post I'll be talking about how each of these are used.

Padding is an invisible layer that is just outside of the content. If you increase the padding, the content size will appear to increase for the additional size that the padding adds.

Border is the only visible layer. It adds a visible border around the padding as seen in the above image.

Margin is another invisible layer that is on the outside of the border. Increasing this will appear to push the content away from other elements. You can change side of the margin individually, which is useful for positioning relative to other elements on the web page.
