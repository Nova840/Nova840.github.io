Constructors: Javascript vs Ruby

3/26/16

A constructor is a function that creates a new object when you call it with the new keyword. In JavaScript, it looks like this: 

function thing(property1, property2){
	this.property1 = property1;
	this.property2 = property2;
}

// Create a new object using the constructor.
var newThing = new thing("Value 1", "Value 2");

In Ruby, it's a little different. A constructor will set the initial values of a class. 

class ExampleClass

	def initialize(property1, property2)
		@property1 = property1
		@property2 = property1
	end

end

# Create a new instance of ExampleClass using the constructor.
instance = ExampleClass.new("Value 1", "Value 2")
