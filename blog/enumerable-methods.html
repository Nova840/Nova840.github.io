Using the map method in Ruby.

3/10/16

The map method in Ruby is an enumerable method in ruby that lets you modify every element in an array. It's just like the each method except it will change the array to whatever the element is assigned to. If there is no assignment, the element will be assigned to nil. 

this_array.map {|i| i = new_value }aaaaaaaaaaaaaaaaaaaaa

The above method would return an array with this_array's length and every value in the array would be equal to new_value.
