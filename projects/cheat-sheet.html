While loops

A while loop executes a block of code while a condition is true. 

while(condition){
	//code block
}

A do-while loop is almost the same as a while loop, but the condition check happens after the code block executes. This means the block always executes at least once. It's functionally the same as retyping the code block above a standard while loop. 

do{
	//code block
}
while(condition);

For loops

For loops in JavaScript can be written in multiple ways. The first is the standard for loop: 

for(var i = 0; i < array.length; i++){
	console.log(array[i]);
}

The next way is to use a for-in loop. This works just like the standard for loop, but it's less to type. Note that "i" is still the index, not the element itself. 

for(var i in array){
	connsole.log(array[i]);
}

There is a way to get the element itself as opposed to the index. For that you would need a for-each loop. 

array.forEach(function(element){
	console.log(element);
});
